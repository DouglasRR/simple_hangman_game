# Jogo da Forca em Python

Este é um jogo da forca simples em Python, onde o jogador deve adivinhar uma palavra oculta antes de esgotar todas as chances. O jogo possui uma interface de linha de comando e pode ser uma ótima maneira de praticar suas habilidades em Python.

## Funcionalidades

- O jogo seleciona aleatoriamente uma palavra de uma lista predefinida.
- O jogador tem um número limitado de chances para adivinhar a palavra.
- As letras corretamente adivinhadas são reveladas na palavra oculta.
- O jogador é informado sobre letras incorretas e o estágio atual da forca.

## Como Jogar

Para jogar o jogo da forca, siga estas etapas:

1. Clone ou faça o download deste repositório.

2. Certifique-se de ter o Python instalado em seu computador.

3. Execute o arquivo `forca.py` para iniciar o jogo.

4. Digite letras para adivinhar a palavra oculta.

5. Continue adivinhando até adivinhar a palavra ou até ficar sem chances.

## Requisitos

- Python 3.x
